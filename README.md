## Setup Requirements

1. Download [FFmpeg Essentials Build](https://www.gyan.dev/ffmpeg/builds/).
2. Extract it.
3. Place `ffmpeg.exe` in the same folder as your scripts OR add its `bin` folder to your system PATH.
